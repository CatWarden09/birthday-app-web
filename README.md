## Краткое описание

Поздравлятор - веб-приложение для хранения, отображения и управления списком дней рождения.

Основные возможности:

- Отображение сегодняшних, ближайших и прошедших (+ и - неделя соответственно) дней рождения
- Отображение общего списка дней рождения
- Автоматический подсчет возраста именинника на момент ДР для отображения в таблице
- Автоматическая сортировка общего списка ДР по возрастанию даты рождения
- Добавление, удаление и редактирование дней рождения
- Загрузка, отображение и изменение фотографий именинников
- Автоматическое удаление файла фотографии из рабочей директории проекта при удалении ДР или замене фотографии
- Отдельное редактирование фото и имени/даты (фото не меняется/не удаляется, если не загружать новое)
- Валидация даты рождения при создании/редактировании ДР (не может быть в будущем или больше 120 лет назад)
- Навигация между страницами сервиса при помощи меню

## Конфигурация
Для корректной работы программы необходимо:

- Создать БД в PostgreSQL. Таблица birthday создается автоматически при запуске программы
- Создать в корневой директории исходного проекта/собранного проекта (см. раздел [запуск проекта](#запуск-проекта)) файл config.properties и указать в нем следующие параметры:

```
db.url=jdbc:postgresql://<хост>:<порт>/<имя_БД>
db.user=<имя_пользователя> (по умолчанию postgres)
db.password=<пароль>
```

Пример содержимого файла config.properties:
```
db.url=jdbc:postgresql://localhost:5432/birthdays
db.user=postgres
db.password=admin
```
## Запуск проекта

Проект можно запустить из любой IDE с поддержкой Maven (например, IntelliJ IDEA).

Для запуска проекта вне IDE необходимо провести сборку:

1. **Собрать проект**

Собрать проект можно через консоль или напрямую через IDE (перед этим нужно указать в PATH путь к Maven, если еще не указан и загрузить на ПК сам [Maven](https://maven.apache.org/download.cgi#/CurrentMaven), если еще не загружен :) )

Вариант через консоль:

- Выполнить команду в корневой директории проекта:
  
```
mvn clean package
```

2. **Настроить конфигурацию**

В подпапке `target/` (собранный проект) нужно создать файл конфигурации `config.properties` и заполнить его в соответствии с [инструкцией](#конфигурация) выше

3. **Запустить проект**

В папке `target/` выполнить команду запуска:

```
java -jar *birthday-app-web-1.0.jar*
```

4. Открыть сервис в браузере по адресу localhost:8080 (порт по умолчанию, настраивается в application.properties)
